## 🔍 已知限制

1. **数据不持久化** - 应用关闭后数据清空（设计行为）
2. **设备识别** - 仅通过 IP 地址识别设备
3. **单个 JSON 大小** - 建议不超过 1MB
4. **并发限制** - 受 Dart HttpServer 默认限制

## 💡 使用提示

```bash
# 启动应用
```

### 生产环境发送数据
```bash
# 使用 curl
curl -X POST http://<服务器IP>:10010/event \
  -H "Content-Type: application/json" \
  -d '{"your": "data"}'

# 使用 Python
import requests
requests.post("http://<服务器IP>:10010/event", json={"your": "data"})
```

```bash
curl http://<服务器IP>:10010/
```
